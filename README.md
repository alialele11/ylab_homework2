# ylab_homework2
##Игра Обратные крестики-нолики с правилом "Пять в ряд"
Правила игры:
Вы ходите первые "Х", далее ходит компьютер "О". Игра продолжается до тех пор, пока не определится проигравший. Проигравшим считается игрок, который соберет по вертикали, по горизонтали или по диагонали 5 своих символов в ряд.

#Инструкция по запуску
1. Запустите файл game.py
2. Для того, чтобы сделать шаг укажите сначала номер строки, куда хотите поставить, от 0 до 9, затем укажите номер столбца от 0 до 9
3. Дождитесь, пока компьютер сделает свой шаг
4. Прдолжайте игру, пока не определится победитель или не будет ничья
